# Stick War Legacy Mod Menu (UI Simulation)

[![Mod Menu Version](https://img.shields.io/badge/Version-V2.0-blue)](https://github.com/yourusername/stick-war-mod)
[![License](https://img.shields.io/badge/License-MIT-green)]()

Đây là giao diện mô phỏng (Simulation UI) một Mod Menu dành cho game Stick War Legacy. Được viết bằng HTML, CSS và JavaScript thuần. 

> ⚠️ **Lưu ý:** Đây chỉ là giao diện giả lập dùng để quay video, chụp ảnh hoặc làm Project cá nhân. Nó **không** có khả năng can thiệp vào bộ nhớ của game thật trên iOS/Android.

## ✨ Các tính năng nổi bật
- **Modded By MKgamer:** Logo tên tuổi với hiệu ứng cầu vồng chuyển động.
- **Profile:** Free Shopping, Unlitimate Point Update, Max Update, Unlock all unit, Unlitimate Gems, Unlitimate Golds.
- **Spawn:** Menu thả (Dropdown) chứa 10 loại quân (SwordWrath, Spearton, Giant, Finalboss...) + Nút Spawn unit.
- **Player:** Fast Build, Unlitimate Population.
- **Enemy:** Fast Build, Unlitimate Population.
- **Game Manager:** Control Zombie, Auto Win, Kill enemy, Kill Player.
- **Spell:** Dropdown chứa 7 loại Spell (SwordWrath Rage, Rain Heath...) + Nút Spawn Spell.
- **Statue:** Statue Unlitimate Heath, Statue laser eyes.
- **Gems:** Thanh trượt (Slider) chỉnh số lượng Gems từ 1 đến 1.000.000.
- **TikTok:** Nút chuyển hướng nhanh đến trang TikTok cá nhân.
- **Hack Status:** 2 dòng chữ "HACK DISABLED" / "HACK ENABLED" ở đáy trái phải, hiệu ứng cầu vồng.

## ⚙️ Tính năng ẩn (Settings)
Bấm vào nút **Bánh răng (⚙️)** ở góc trên bên phải để ẩn toàn bộ menu chính, hiện ra mục **LETTER**.
- Kéo thanh trượt "Letter Size" để **thay đổi kích thước toàn bộ chữ** trong menu theo thời gian thực.
- Bấm nút **X** màu đỏ để quay lại menu chính.

## 🚀 Cách sử dụng
1. Tải xuống (Download) file `index.html` và `Anh.jpg`.
2. Đặt 2 file này nằm cùng một thư mục trên máy tính/điện thoại.
3. Mở file `index.html` bằng trình duyệt (Chrome, Cốc Cốc, Safari...).
4. Giao diện Mod Menu sẽ hiện ra ngay lập tức.
